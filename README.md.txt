하나는 c언어 코드파일 이며 다른 하나는 c++코드파일 입니다. 비교를 위해 두가지 파일 모두 첨부하였습니다!

<코드 속 c++특징>

(1). 코드를 보면 구조체 대신 piece와 board라는 클래스를 사용하여 게임의 판과 움직이는 조각을 각각 독립된 단위로 묶어서 캡슐화를 진행했습니다.

(2). public: 이라는 접근 제어 지시자를 사용하여 외부에서 클래스의 멤버함수에 접근할 수 있도록 해주었습니다.

(3). pass by reference를 이용하여 참조자를 매개변수로 받아 객체 전체를 복사하지 않고, 원래 메모리 주소에 그대로 전달하므로 성능면에서 좋게 작성할 수 있습니다.

(4). 기본 자료구조 형태인 2차원배열을 사용하여 코드를 작성하였으며, 이를 통해 배열 기반의 데이터 구조를 엿볼 수 있습니다.

(5). c++의 절차적 특징과 실시간 그래픽 프로그래밍의 특징을 조합하여 게임에 녹여냈습니다.


요약하자면, 클래스를 통해서 데이터와 기능을 하나로 묶어서 관리하고, 캡슐화를 통해서 객체 내부의 로직을 독립적으로 관리했으며, 참조를 통해서 메모리 낭비를 줄이면서 정보를 전달하고, 마지막으로 초기화를 통해서 객체가 생성될 때 멤버 변수 값이 설정되게 구성되어있습니다.

<c언어와 차이점>
1. 구조체 -> 클래스
2. 독립함수 -> 메서드
3. c언어 표준 stdbool.h 헤더파일 사용
